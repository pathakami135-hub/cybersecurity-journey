## Cybersecurity Roles Learned

### Security Analyst (Blue Team)
- Monitors systems and networks.
- Investigates security alerts.
- Responds to security incidents.

### Security Engineer
- Designs and maintains security systems.
- Configures IDS and other security tools.
- Protects systems from vulnerabilities.

### Penetration Tester (Ethical Hacker)
- Tests systems for security weaknesses.
- Finds vulnerabilities before attackers do.
- Provides recommendations to improve security.

## Key Takeaways
- Cybersecurity offers many career paths.
- Blue Team focuses on defense.
- Penetration testing is authorized ethical hacking.
- Security Engineers build and maintain secure environments.
